# 彭铿个人学术网站 - 产品需求文档

## 1. 产品概述
为彭铿(Keng Peng)创建一个专业的个人学术主页网站，用于展示学术背景、研究成果、发表论文和联系方式。
- 目标用户：学术同行、潜在合作者、学生、招聘方
- 产品价值：建立专业的学术形象，便于他人了解研究工作并取得联系

## 2. 核心功能

### 2.1 功能模块
1. **首页/关于我**：个人照片、姓名、职位、所属机构、简短自我介绍、研究兴趣、邮箱联系方式
2. **教育背景**：教育经历时间线展示
3. **研究成果**：发表论文列表，可按年份/类型分类
4. **荣誉奖项**：获奖情况展示
5. **联系我**：联系表单或联系方式卡片

### 2.2 页面详情
| 页面名称 | 模块名称 | 功能描述 |
|----------|----------|----------|
| 首页 | Hero区域 | 展示个人照片、姓名、职位、机构、邮箱，背景采用学术风格设计 |
| 首页 | 导航栏 | 固定顶部导航，平滑滚动到各锚点，中英文切换支持 |
| 首页 | 关于我 | 个人简介、研究方向关键词标签 |
| 首页 | 教育背景 | 时间线形式展示学历信息 |
| 首页 | 研究成果 | 论文列表，包含标题、作者、期刊/会议、年份、链接 |
| 首页 | 荣誉奖项 | 卡片式展示获奖经历 |
| 首页 | 联系我 | 邮箱链接、机构地址等联系方式 |
| 首页 | 页脚 | 版权信息、最后更新时间 |

## 3. 核心流程
用户访问网站后，可以通过导航栏快速跳转到各个部分浏览，点击邮箱可直接发送邮件，论文链接可跳转至外部页面。

## 4. 用户界面设计

### 4.1 设计风格
- **主色调**：深蓝/藏青色 (#1e3a5f) 作为学术专业色调，搭配金色/琥珀色 accents (#c9a227) 增加高级感
- **辅助色**：浅灰、白色作为背景，深灰作为正文
- **按钮样式**：简约圆角设计，hover时有微妙阴影和颜色过渡
- **字体**：标题使用衬线字体(如 Playfair Display)体现学术气质，正文使用无衬线字体(如 Source Sans Pro)保证可读性
- **布局风格**：单页滚动式设计，左侧/顶部导航，卡片式内容组织，充足留白
- **图标风格**：简约线条图标

### 4.2 页面设计概览
| 页面名称 | 模块名称 | UI元素 |
|----------|----------|--------|
| 首页 | Hero区域 | 全屏高度，左侧文字右侧头像/占位图，渐变背景，文字逐行淡入动画 |
| 首页 | 导航栏 | 固定顶部，毛玻璃效果，滚动时背景透明度变化 |
| 首页 | 内容区块 | 卡片式设计，微妙阴影，hover时轻微上浮效果 |
| 首页 | 时间线 | 左侧竖线+节点，右侧内容，优雅的连接线 |
| 首页 | 论文列表 | 按年份分组，每篇论文卡片hover有边框高亮 |

### 4.3 响应式设计
- 桌面优先设计，适配平板和手机
- 移动端导航转为汉堡菜单
- 卡片在小屏幕上垂直堆叠
- 字体大小和间距自适应调整

### 4.4 动效设计
- 页面加载时各区块依次淡入
- 导航点击平滑滚动
- 卡片hover时有微妙变换
- 滚动时导航栏样式变化
